# Nelavai Hema Portfolio

A full-stack portfolio website built with React, Tailwind CSS, and Express.

## Setup

1. Install dependencies:
   ```bash
   npm install
   npm install --prefix client
   npm install --prefix server
   ```

2. Create server environment variables using `.env` or `.env.example`.

3. Run the application:
   ```bash
   npm run dev
   ```

## Available scripts

- `npm run dev` - Start client and server together.
- `npm start` - Start the Express server (build the React app first).
- `npm run build` - Build the React client.
